Select the service you want to subscribe for your project

A service is a set of features for your project. They may differ depending on the available tool you will choose.

Select a tool providing the selected service.

A tool is specific implementation of a service, and several instances (or nodes) can be deployed.

Select a node executing the tool.

A node is a running instance, server or whatever instance based of a tool.

Select the creation mode for this subscription. More information on selection.

A mode is a type of subscription. "Link" mode will link the project to the tool, and "Create" will in addition, create the project inside the tool.

Provide required parameters for the selected node.

Parameters are used to maintain the association between the project and the node. In addition, in creation mode, parameters are required to create the project inside the target node.
